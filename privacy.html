Politica de Privacidade

Ultima atualizacao: 22 de marco de 2026

1. Dados coletados

Para funcionamento do bot e do dashboard, a Luma pode processar dados tecnicos do servidor e do uso de comandos.

  • ID de servidor, ID de canal, ID de cargo e configuracoes de modulos.
  • Registros operacionais de moderacao, tickets e modmail, quando habilitados.
  • Dados basicos de autenticacao necessarios para sessao no dashboard.

2. Finalidade do uso dos dados

Os dados sao utilizados para manter recursos do bot funcionando, fornecer historico operacional e permitir administracao do servidor via dashboard.

3. Compartilhamento

A Luma nao vende dados pessoais. Dados podem ser processados por provedores de infraestrutura e servicos necessarios para operacao tecnica.

4. Retencao e seguranca

Os dados sao mantidos apenas pelo tempo necessario para operacao, auditoria e seguranca do servico, respeitando limites tecnicos e legais aplicaveis.

5. Direitos e solicitacoes

Donos de servidor podem solicitar esclarecimentos sobre dados do proprio servidor e ajustes quando tecnicamente viavel, usando os canais oficiais da comunidade da Luma.

6. Atualizacoes desta politica

Esta politica pode ser atualizada para refletir mudancas tecnicas, operacionais ou legais. A versao mais recente sera publicada nesta pagina.
